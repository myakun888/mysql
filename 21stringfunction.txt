//! STRING FUNCTION

di mysql juga memilki string function, sehingga kita bisa memanipulasi string di mysql diantaranya

lower()  --> mereturn string menjai huruf kecil
upper()  --> merubah string menjadi huruf besar
lcase (), ucase() persamaan atau ganti dari lower dan upper ,fungsi ny sama


concat() --> menggabungkan string
concat_ws() --> menggabungkan string dengan tanda sparator / pemisah
Ltrim()  --> untuk memotong / membuang spasi di awal sebelah kiri
Rtrim()  --> untuk memotong / membuang spasi di sebelah kanan
trim()   --> untuk membuang spasi di kanan dan kiri string
